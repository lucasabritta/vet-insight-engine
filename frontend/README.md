# Frontend

React + TypeScript frontend for the Vet Insight Engine.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Runs on http://localhost:5173

## Building

```bash
npm run build
```

## Testing

```bash
npm test
npm run test:coverage
npm run test:ui
```

## Linting

```bash
npm run lint
npm run type-check
```
